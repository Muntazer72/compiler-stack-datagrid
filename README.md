This project is a web-based implementation of core concepts required for designing a compiler. It focuses on managing a **Symbol Stack** and visualizing data through a dynamic interface that mimics the `DataGridView` found in C# environments.

### Lab Objectives:
1. **Dynamic Data Grid:** Displaying and retrieving values from a structured table.
2. **Run-time Insertion:** Ability to add tokens/values to the UI at runtime.
3. **Stack Implementation:** A functional LIFO (Last-In, First-Out) data structure to manage compiler states.
